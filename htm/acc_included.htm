Included Accessories

Several topside accessories are included with all MSSPro 4 Ultra Lite system configurations.

Sun Shade

The sun shade can beiscan be attached directly to the control panel lid and provides shade for the computer and monitor to make it easier to see the displays when working in bright light. Tool Kit

A basic tool kit is provided in order to perform routine maintenance and field repairs. The tool kit also contains some spare parts including ballast weights, propellers, O-rings and other items.
