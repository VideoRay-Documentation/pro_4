Lyyn Hawk

The LYYN visibility enhancement systems provide real time image clarification allowing you to see more details and further in water with limited visibility. They can be used to process the video directly from the Pro 4 Ultra, or with any external composite video source.

LYYN HawkTM

The LYYN Hawk board can be built into the control panel to eliminate the need to carry additional components or connect cables.

Early models of the Pro 4 Ultra control panel included a membrane keypad to control the LYYN functions. VideoRay Cockpit software can now be used to control the LYYN functions. See the LYYN documentation for information about using the keypad and the section in the VideoRay Cockpit Guide for more information about the software controls.

LYYN T38TM

The LYYN T38 is a self contained version of the LYYN Hawk system.
