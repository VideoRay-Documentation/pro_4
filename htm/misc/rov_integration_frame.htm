ROV Integration Frame

The ROV Integration Frame is a custom attachment designed to carry the following accessories:

  • BlueView ProViewer Sonar
  • VideoRay Lateral Thruster
  • VideoRay Laser Scaling Device
  • SeaSprite Micron Echosounder (Altimeter)
  • Long Baseline Transponder

The Integration Frame features a tilting BlueView ProViewer sonar mount, adjustable ballast and a trim fin for optimizing vertical dynamic trim.

See the Accessories Guide for more information about using each accessory.
