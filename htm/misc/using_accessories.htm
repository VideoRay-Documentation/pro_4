Using ROV Mounted Accessories

ROV mounted accessories generally require the following steps:

  1. Mount the accessory on the ROV.
  2. Connect the power/interface cable to the ROV accessory port.
  3. Test the accessory for proper function.
  4. Ballast and trim the ROV.
  5. Commence the dive operations.

Exercise

  1. Attach the manipulator to the ROV using the clamps provided.
  2. Connect the cable to the ROV accessory port.
  3. Test the manipulator function.

Tips

Always make sure the accessory port connection is sealed with a accessory port dummy plug or manipulator or cutter.

Make sure to complete the full Pre-dive checklist, not just the checklist for the accessory.
