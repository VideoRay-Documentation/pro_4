Pro 4 Ultra Lite Notes

The Pro 4 Ultra Lite has the following differences from the Standard Pro 4:

VideoRay Cockpit Software

  • Instrument display is fixed
  • Water Temperature instrument is not available
  • Turns Counter instrument is not available
  • User Defined text overlay on video display is not supported
  • Data Import is not supported
  • Data import as text overlay on video display is not supported
  • Data Export is not supported
  • Network Remote operation is not available
  • Engine Room tuning is not supported
  • WMV is the only video recording format supported
  • No real time audio recording
  • Video overlay text is not supported
  • Camera focus adjustment is disabled
  • Camera selection for external camera is disabled

Hardware

ROV

  • No differences

Control Panel

  • One AC power pass-through instead of two
  • One USB connection instead of three
  • No 12 V DC output
  • Tether voltage is 48 V DC, not 74 V DC
  • No LIM (not required)
  • No second display monitor support
  • Integral sunshade not included

Computer

  • 2 USB ports
  • No Ethernet port
  • Computer not included (user must supply own their computer)

Tether Compatibility

  • No Differences

Accessory Support

  • All accessories that are supported on the Pro 3 are supported, except KCF Smart Tether
  • Pro 3 and Pro 4 Smart Tether are not supported
  • PAM based accessories are not supported
  • No control panel internal accessory interfaces are available
